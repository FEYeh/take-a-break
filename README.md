# take-a-break

> A simple CLI tool for setting an alarm.

_**notice**: the computer will play a [beep sound]('./src/media/analog-watch.mp3') when time's up._

## Run

```shell
$ npx take-a-break
```

or

```shell
$ npm i take-a-break -g
$ take-a-break
```
